# 📚 DigitalShelf - Library Management System

## Team 6 BITS | Gordon College | BSIT 1st Year

### Project Overview
DigitalShelf is a desktop-based Library Management System built with Java Swing and MySQL. It allows students to browse, borrow, and return books, while administrators can manage books, students, and view all transactions.

### Technologies Used
- **Frontend:** Java Swing
- **Backend:** Java (JDK 25)
- **Database:** MySQL
- **Connector:** JDBC

### Features

#### Student Features
- Browse books with cover images
- Search books by title/author
- Borrow books (14-day due period)
- Return books
- View borrowing history

#### Admin Features
- Manage books (Add/Edit/Archive/Unarchive)
- Manage students (Add/Delete)
- View all borrow records
